# POS System Web

Sistema POS web adaptable para diferentes tipos de negocios como restaurantes, cafeterías y tiendas.

## Tecnologías

Frontend

* Angular
* Bootstrap

Backend

* .NET Web API

Base de datos

* PostgreSQL

## Funcionalidades iniciales

* Gestión de productos
* Registro de ventas
* Control de inventario
* Reportes de ventas
* Gestión de usuarios
